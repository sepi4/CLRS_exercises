selection sort

```cpp
selectionSort(A)
    n = A.length
    for i = 1 to n - 1                          n - 1
        iSmallest = i
        for j = i + 1 to n                      (n - 1)^2
            if A[j] < A[iSmallest]
                iSmallest = j
        swap(A[iSmallest], A[i])
```
